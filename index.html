About me

Hi, my name is Denis and I'm a "Junior Full-Stack Web Developer". I was born and raised in Croatia in 1991. I started developing my interest in coding around 2015 because I was dissatisfied with my career and occupation at the time. It was one of several life-changing decisions in my life and one that I have never regretted. I find coding and software development quite challenging, enjoy working on projects with other people, solving and optimizing issues in coding.

The first programming language I started learning was PHP and with it HTML and CSS languages. Not long after I started working on my first projects using PHP I was offered the opportunity to learn C# including the ASP.NET framework for web application development. After finishing that course in Croatia, I decided to leave my home country and move to Munich/Germany in 2019. In September 2023, I attended the "Full-Stack Web Developer" course to expand my previous knowledge and challenge myself even more. In addition, I would say that I am very ambitious and always ready to learn more.
